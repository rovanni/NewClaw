Primeira vez? Comece por Config antes de testar qualquer tool.

🚀 Passo 1 — Configurar o Sistema

Abra Config e configure:

  • Provider — Ollama, Gemini, DeepSeek ou Groq
  • Modelo — selecione ou digite (ex: glm-5.1:cloud)
  • Idioma — pt-BR, en-US ou es-ES
  • Tools — ative/desative ferramentas
  • Salvar & Reiniciar — aplique mudanças

💬 Passo 2 — Conversar no Chat

Na tela Chat:

  • + Nova — cria conversa vazia
  • 🎤 — reconhecimento de voz do navegador
  • 🔊 — liga/desliga TTS automático
  • 📥 / 📤 — exporta/importa conversas em JSON
  • 📋 — copia resposta individual (hover na mensagem)
Enter envia, Shift+Enter cria nova linha.

🔍 Passo 3 — Auditar com Traces

Na tela Traces, veja o caminho de cada execução:

  • Decisão — como o agente classificou a intenção
  • Tool Calls — quais ferramentas foram chamadas
  • Resultados — o que cada tool retornou
  • Resposta Final — texto enviado ao usuário
Se o chat respondeu errado, olhe o trace antes de mexer no prompt.

🖱️ Interações no Grafo

Selecionar Nó

Clique em qualquer nó. Os outros ficam semi-transparentes, o selecionado ganha destaque com borda azul. O painel direito mostra detalhes, relações e edição.

Criar Relação

Modo Conectar: Clique 🔗 Conectar → clique nó origem → clique nó destino.
Ctrl+clique: Ctrl+clique no nó origem → clique no destino. Esc para cancelar.
Manual: Preencha "De" e "Para" no sidebar esquerdo.

Desfazer (Undo)

Após deletar um nó ou relação, o botão ↩️ Desfazer aparece ativo por 10 segundos com countdown. Clique para restaurar.

Modo Foco

Clique 🔍 Isolar Nó para ver apenas o nó e vizinhos. Controle profundidade (1 hop, 2 hops, tudo) e importância mínima com o slider.

❓ Dúvidas Comuns

O chat respondeu errado. → Abra Traces, verifique a decisão e tool chamada antes de mexer no prompt.

Troquei o modelo e nada mudou. → Use Salvar & Reiniciar na tela Config.

A memória está bagunçada. → Use memory_admin com ação cleanup ou acesse Revisão para mesclar duplicatas.

O grafo está poluído. → Use Modo Foco (isolar nó) + slider de importância. Clique nos tipos na legenda para ocultar categorias.

Deletei um nó por engano. → Clique ↩️ Desfazer em até 10 segundos.

Como conectar dois nós rápido? → Ctrl+clique no nó origem → clique no destino. Ou use o botão 🔗 Conectar.

Posso usar só pelo Telegram? → Sim, mas o dashboard é essencial para config, traces e manutenção.
